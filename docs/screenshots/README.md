# Screenshots Directory

Add your game screenshots here! 

## How to Add Screenshots

1. Take screenshots of your game while it's running
2. Save them with these names:
   - `gameplay-1.png` - Main game interface
   - `gameplay-2.png` - Combat in action
   - `gameplay-3.png` - Card selection
   - Or add more with any names you like!

3. Update the `docs/index.html` file to include your new screenshots in the gallery

## Recommended Screenshot Sizes

- Width: 1200-1600px
- Format: PNG (for best quality) or JPG
- Aspect Ratio: 16:9 or 4:3

## Tips

- Take screenshots showing different aspects of the game
- Show the pixel-art style GUI
- Include examples of card play, combat, and enemy encounters
- Make sure the UI is clearly visible
