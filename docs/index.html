Card Battler is a turn-based deck-building game where you battle enemies using strategic card play. Build your deck, manage your energy, and defeat all enemies to progress through floors!

üöÄ Quick Start

Installation

The game runs on pure Python 3.7+ with no external dependencies required for the desktop GUI!

# Clone the repository
git clone https://github.com/YOUR_USERNAME/card-battler.git
cd card-battler

# Run the desktop GUI (no dependencies needed!)
python3 run_gui.py

# Or run the web version (requires Flask)
pip install flask
python3 web_gui_enhanced.py
# Then open http://localhost:8080 in your browser

üéÆ How to Play

  1. Start Combat: A new combat begins automatically with enemies and your starting hand.
  2. Play Cards: Click on cards in your hand to play them. Each card costs energy.
  3. Manage Energy: You start with 3 energy per turn. Plan your plays carefully!
  4. End Turn: Click "End Turn" when done. Enemies will attack, then your turn begins again.
  5. Win: Defeat all enemies to win the combat and progress to the next floor!

üìñ Game Mechanics

‚ù§Ô∏è Health & Block

Your HP determines survival. Block reduces incoming damage. Both are shown with visual bars.

‚ö° Energy

Energy is spent to play cards. You gain 3 energy per turn. Some cards can generate extra energy.

üÉè Card Types
