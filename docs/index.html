‚ú® Features

üÉè Card System

Data-driven card system with attacks, skills, and powers. Each card has unique effects, costs, and rarities. Cards can deal damage, provide block, draw cards, generate energy, or apply status effects.

‚öîÔ∏è Combat Engine

Turn-based combat with player and enemy turns. Manage energy, block, and status effects strategically. Enemies have intents showing what they'll do next turn.

üé® Beautiful UI

Modern pixel-art style GUI with visual health bars, card animations, color-coded combat log, and intuitive controls. Dark green theme with high-contrast buttons for easy reading.

üëπ Enemy Encounters

Procedurally generated enemy encounters that scale with floor level. Each enemy has unique intents and abilities, creating varied and challenging combat scenarios.

‚ú® Status Effects

Rich status effect system with buffs, debuffs, and damage-over-time effects. Includes Strength, Vulnerable, Weak, Poison, Burn, and many more that add strategic depth.

üåê Multiple Interfaces

Play via desktop GUI (tkinter), web browser (Flask), or command line. Choose what works for you! All interfaces provide the same great gameplay experience.

üöÄ Quick Start

Installation

The game runs on pure Python 3.7+ with no external dependencies required for the desktop GUI!

# Clone the repository
git clone https://github.com/wnorton26-sketch/Winter-term-game.git
cd Winter-term-game

# Run the desktop GUI (no dependencies needed!)
python3 run_gui.py

# Or run the web version (requires Flask)
pip install flask
python3 web_gui_enhanced.py
# Then open http://localhost:8080 in your browser

üéÆ How to Play

  1. Start Combat: A new combat begins automatically with enemies and your starting hand.
  2. Play Cards: Click on cards in your hand to play them. Each card costs energy (shown as orange badges).
  3. Manage Energy: You start with 3 energy per turn. Plan your plays carefully!
  4. End Turn: Click the bright green "END TURN" button when done. Enemies will attack, then your turn begins again.
  5. Win: Defeat all enemies to win the combat and progress to the next floor!

üìñ Game Mechanics

‚ù§Ô∏è Health & Block

Your HP determines survival. Block reduces incoming damage. Both are shown with visual bars and color-coded indicators (green = healthy, orange = warning, red = danger).

‚ö° Energy

Energy is spent to play cards. You gain 3 energy per turn. Some cards can generate extra energy, allowing for powerful combo turns.

üÉè Card Types

Attack: Deal damage to enemies
Skill: Gain block or apply effects
Power: Persistent effects that last all combat

‚ú® Status Effects

Strength: +1 damage per stack
Vulnerable: Take 50% more damage
Weak: Deal 25% less damage
Poison: Damage at start of turn
And many more!
